# Install xSuite

The only requirement is to have Node.js 18+ already installed.

<details>
  <summary>Install Node.js 18+</summary>

Install the [Node Version Manager](https://github.com/nvm-sh/nvm/tree/v0.39.5#install--update-script) (NVM):

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
```

Then install Node.js 18:

```
nvm install 18
```

<Callout type="warning" emoji="⚠️">
  You should then close and reopen your terminal.
</Callout>

</details>

To install xSuite:

```
npm install -g xsuite
```
